本站说明

Posted on 2015-08-29   |     |  

经过几天的研究，基于 Hexo 和 GitHub 的此博客终于建立起来。经过多次测试和修改，仍然有几个问题未能解决。

  • 搜索功能无法正常使用。当使用搜索时，点击搜索的结果，理论上会跳转到文章的详细页，然而实际上是失败的。原因是跳转时链接前面的“www.”丢失，由于本人愚钝，至今未找到解决方案，若有知晓者还望不吝赐教。
  • RSS 订阅无法使用。虽然 RSS 已经被淘汰了，但是，有个东西叫“情怀”，你懂的。
  • 界面过于简陋。由于专业知识的欠缺，不会做 Web 前端，以后若有时间，会做改变。
  • 其他问题暂未发现。如果您在浏览本站的过程中发现问题，请在本文下方留言，我会在第一时间回复，谢谢。

如果您喜欢本站的文章，欢迎转载和分享，谢谢。

再次感谢。

我是Lu_Sir，希望你开心。

坚持原创技术分享，您的支持将鼓励我继续创作！
